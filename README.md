# Understanding Support Vector Machines: How Kernel Functions Shape Decision Boundaries

This repository contains the materials for a machine learning tutorial that demonstrates how different kernel functions influence the decision boundaries of Support Vector Machines (SVMs).

## Tutorial Overview

Support Vector Machines are powerful classification algorithms that identify optimal decision boundaries between classes. Kernel functions allow SVM models to capture nonlinear relationships by transforming the data into higher-dimensional spaces.

This tutorial compares three commonly used kernel functions:

- Linear Kernel
- Polynomial Kernel
- Radial Basis Function (RBF) Kernel

The behaviour of these kernels is demonstrated using the Breast Cancer dataset from Scikit-learn.

## Requirements

To run the notebook, install the following Python libraries:

- numpy
- pandas
- matplotlib
- scikit-learn
